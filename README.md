# gotomusic  
create a landing page for your music  

## usage  

### command line  
#### Clone repository  
>git clone https://github.com/bekatru/gotomusic.git
  
#### Enter the repo directory  
>cd gotomusic  
  
#### Install dependencies  
>npm install  
  
#### Start local host  
>start live-server  
  
#### **or**

#### Open the index.html  
>start index.html
